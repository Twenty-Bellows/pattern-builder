Manage Patterns in the WordPress Edito

== Development ==

The plugin source is available here: https://github.com/Twenty-Bellows/pattern-builder

Node & NPM are needed to install and run the development tools:

```
npm install
npm run build
npm run watch
```

Docker is needed to run the development environment
```
npm start
npm stop
```

See the source for more details.
